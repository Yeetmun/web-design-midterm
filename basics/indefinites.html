Indefinite Integrals

What is an integral?

You've (hopefully) already learned all about derivatives. Integrals are simply derivatives in reverse, or an antiderivative just like how subtraction is basically additions but in reverse. Another way to think about is is by asking "If a function's derivative is this, what is the original function?" This is very handy if you say, wanted to find the velocity function of something but only have the acceleration function. Since acceleration is just the derivative of velocity, if we take the antiderivative, or integral, of acceleration then we end up with velocity.

But you do you write the integral? To do that you use the symbol ∫ to denote the start of an integral. You then insert the function you want to be integrated such as f(x) after the integral symbol. Finally, you need to add dx in order to complete the integral. Be careful though, it doesn't necessarily have to be dx, it can be any variable but is usually the variable used in your function.


Solving Integrals

Now that you've learned what integrals are, let's solve a simple practice problem. Take the function 2x for example, and we want to find the integral, or antiderivative, of it. Using our symbols from earlier, that would look something like this ∫2xdxRemember, we're solving for something that, when we apply the derivative, results in 2x. Obviously that means the answer is x2, but there's one more thing we have to add, and that's a +C. This is because derivatives eliminate any constants so we can't know if there was a constant before or not after integrating. As such we have to add a constant to represent all these possible values.


What does it mean?

What does the integral represent in the real world? That's quite simple, it simply represents the area of a curve or function between that function or curve and the x-axis. This property can lead to many useful applications in physics, engineering, medicine, economics, and more. One such example could be determing the volume of a shape that's complex to measure but has an easily modelable surface.
